# Images Folder

Place your location images here!

## How to use:

1. **Add your images** to this folder (e.g., `madrid.jpg`, `barcelona.png`, etc.)

2. **Update constants.tsx** to reference them:
   ```typescript
   images: ['/images/madrid.jpg', '/images/madrid-2.jpg']
   ```

3. **Or use the in-app editor** (Ctrl+E / Cmd+E):
   - Click "Upload" button to upload images directly
   - Or use "Link URL" for hosted images

## Image naming suggestions:
- Use location IDs: `madrid.jpg`, `barcelona.png`, `waterloo.jpg`
- Or descriptive names: `madrid-tapas.jpg`, `barcelona-sagrada.jpg`
